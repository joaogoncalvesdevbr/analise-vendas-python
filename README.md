# 📊 Análise de Dados de Vendas com Python

## 📌 Sobre o projeto
Este projeto realiza uma Análise Exploratória de Dados (EDA) utilizando um conjunto de dados sintético de vendas. O objetivo é identificar padrões, gerar insights e apresentar os resultados por meio de gráficos e estatísticas.

## 🚀 Tecnologias utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📈 Funcionalidades
- Limpeza e tratamento dos dados
- Estatísticas descritivas
- Geração de gráficos
- Exportação de resultados
- Relatório executivo

## 📂 Estrutura do projeto
- Código em Python
- Base de dados
- Gráficos gerados
- Relatório

## 🎯 Objetivo
Demonstrar conhecimentos em Python e análise de dados, aplicando boas práticas de programação e visualização de informações.
